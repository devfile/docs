= Adding schema version to a devfile

[role="_abstract"]
The `schemaVersion` attribute is mandatory in a devfile.


.Procedure

* Define the `schemaVersion` attribute in the devfile:

.Adding schema version to a devfile
====
[source,yaml]
----
v2.0
---
schemaVersion: 2.0.0
metadata:
  name: devfile-sample
  version: 2.0.0
----
====
