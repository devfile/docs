Adding a command section to a devfile

Specify the commands you want to run in your workspace.

Prerequisites

  • Adding schema version to a devfile

  • Adding a name to a devfile

  • Adding projects to a devfile

Procedure

  1. Add a commands section to a devfile that contains a list of one or more commands.

  2. For each command, define a unique value for the mandatory id attribute.

  3. For each command, define a mandatory kind of one of the following kinds:

    • exec

    • apply

    • composite

    Sample command
     commands:
   - id: build
     exec:
      component: mysql
      commandLine: mvn clean
      workingDir: /projects/spring-petclinic
